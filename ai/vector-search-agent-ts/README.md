# AI Agent with Vector Search Sample using TypeScript and Azure DocumentDB

This sample demonstrates building an intelligent AI agent using TypeScript with LangChain and Azure DocumentDB vector search capabilities. The agent provides hotel recommendations based on user queries by leveraging a two-agent architecture: a planner agent for query refinement and vector search, and a synthesizer agent for comparative analysis of results.
